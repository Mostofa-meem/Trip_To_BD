# Trip_To_BD
it's a very simple project using c++, file ,function and classes.
there are three kinds of users 
1. MAIN ADMIN
2. BUS OPERATOR
3. BUS USERS

MAIN ADMIN : admin add bus operator. can add new new route and set bus starting time , can check user info.
Bus Operator : bus operator uses those route and can add bus to a spacific route, can check user info who is using their buses.
user : seat query, seat booking.
